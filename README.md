## 2. 项目说明

本项目由 wpf for .net5 强力驱动

数据来源 [铅笔小说网](http://www.qb123.net)

软件更新服务器, 感谢[永硕E盘](http://www.ys168.com/)提供服务

项目第三方库使用情况
- Caliburn.Micro 4.0.173
- HandyControl 3.1.0
- RestClient 106.11.7
- zipstorer 3.6.0
- HtmlAgilityPack 1.11.33
- System.ComponentModel.Composition 5.0.0

感谢以上第三方库开发人员
